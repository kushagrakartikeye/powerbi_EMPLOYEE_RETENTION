## 📌 Project Overview
This project focuses on analyzing employee attrition using data analytics and visualization techniques. The objective is to identify key factors influencing employee retention and provide actionable insights using **Power BI** and **Python**.

---

## 🎯 Objectives
- Analyze employee attrition trends  
- Identify factors affecting employee retention  
- Perform data cleaning and preprocessing using Python  
- Build an interactive dashboard using Power BI  
- Generate insights to support HR decision-making  

---

## 🛠️ Tools & Technologies
- **Python (Pandas)** – Data cleaning & preprocessing  
- **Power BI** – Data visualization & dashboard  
- **Microsoft Excel / CSV** – Dataset handling

## 🔄 Workflow
1. **Data Collection**  
   - Dataset sourced from IBM HR Analytics dataset (Kaggle)

2. **Data Cleaning (Python)**  
   - Removed duplicates  
   - Handled missing values  
   - Created new features:
     - Salary Slab  
     - Attrition Flag  

3. **Data Visualization (Power BI)**  
   - Built interactive dashboard  
   - Added filters (Gender, Department, Education)  
   - Created multiple visual insights  

4. **Insight Generation**  
   - Identified trends and patterns affecting employee attrition  

---

## 📊 Dashboard Features
- KPI Metrics (Total Employees, Attrition Rate, Active Employees)  
- Attrition by Department  
- Salary vs Attrition Analysis  
- Age Group Analysis  
- Scatter Plot (Income vs Experience)  
- Interactive Slicers

## 🔍 Key Insights
- 📉 Highest attrition observed in the **Sales department**  
- 💰 Employees with **lower salaries** are more likely to leave  
- ⏳ **Early-career employees** show higher attrition  
- 👥 Employees aged **20–30** have higher turnover
